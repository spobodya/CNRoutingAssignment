# Computer Networks Routing Assignment - CNRoutingAssignment

This repository contains the practical implementations for the Computer Networks routing configurations assignment using Cisco Packet Tracer. 

## 📂 Repository Contents

### 1. Task1_Basic_Configuration.pkt
- Hardening of the primary edge router gateway.
- Configured hostnames, custom privileges, and secured line console 0 interactive sessions with explicit authentication.

### 2. Task2_Static_Routing.pkt
- Implementation of a manually mapped triangular multi-router infrastructure (ComputingRouter, BusinessRouter, ScienceRouter).
- Explicit static pathway overrides across localized networks using full next-hop address configurations.

### 3. Task3_Dynamic_Routing.pkt
- Dynamic routing implementation utilizing scale-adaptive algorithms.
- Wiped manual configurations to establish dynamic route updates using Advanced Distance Vector protocol execution.
